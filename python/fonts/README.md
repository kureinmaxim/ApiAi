# Шрифты для ApiAi

Эта папка содержит шрифты TrueType для поддержки кириллицы (если в будущем будет добавлена генерация PDF).

## Содержимое

- `DejaVuSans.ttf` - основной шрифт с поддержкой кириллицы
- `DejaVuSans-Bold.ttf` - жирный вариант

## Установка шрифтов

### Автоматически (Windows/macOS/Linux)

```bash
# Скачать шрифты через скрипт
python scripts/download_fonts.py
```

### Вручную

1. Скачайте DejaVu Sans fonts: https://dejavu-fonts.github.io/
2. Скопируйте `.ttf` файлы в эту папку
3. Убедитесь что есть:
   - `DejaVuSans.ttf`
   - `DejaVuSans-Bold.ttf`

## Для чего нужны

В настоящее время ApiAi не генерирует PDF, но инфраструктура подготовлена на будущее.

Если добавите функцию экспорта результатов в PDF (через ReportLab), эти шрифты обеспечат корректное отображение кириллицы.

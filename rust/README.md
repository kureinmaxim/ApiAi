# ApiAi - Rust Implementation

Экспериментальная версия ApiAi на языке Rust.

## Сборка и запуск

```bash
# Проверка кода
cargo check

# Компиляция
cargo build

# Запуск
cargo run

# Релизная сборка
cargo build --release
```

## Структура проекта

```
rust/
├── Cargo.toml       # Конфигурация проекта и зависимости
├── src/
│   ├── main.rs      # Точка входа
│   └── lib.rs       # Библиотечный код (будет создан)
└── tests/           # Тесты (будет создан)
```
